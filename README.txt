Pour lancer le puissance 4,

	Ouvrir le document index.html
	Une fois que c'est ouvert vous avez les règles du puissance 4 en cliquant sur le boutons à gauche de l'écran
	Pour mettre un jetons appuyez sur les touches 1,2,3,4,5,6,7 (celle situé au dessus de A,Z,E,R,T,Y,U)
	C'est un puissance 4 sur le thèmes de South Park, vous pouvez utilisé soit les jetons de Cartman(rouge), soit ceux de Jimmy(jaune).
